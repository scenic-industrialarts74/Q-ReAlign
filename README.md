## 🚀 Setting Up Your First Task

The application screen shows clear buttons for each step. Follow this process to start your first project.

### 1. Select a Model
You must choose a model size before processing text.
* 0.8B: Best for old computers with little memory.
* 4B: A balance of speed and accuracy.
* 9B: The best quality for complex text tasks.

Click the "Model" tab at the top of the window to pick your preference. The program downloads the file automatically when you select it.

### 2. Prepare Your Data
The software uses text files for input. Create a folder on your computer and place your text documents inside. Make sure these files end in .txt or .csv.

### 3. Run the Workflow
Click the "Workflow" tab. Choose "Select Folder" and point it to the folder you prepared in the previous step. Click the "Start" button to begin processing. You see a progress bar move across the bottom of the window. The program tells you when the task finishes.

## 📂 Managing Your Files

The program uses a system called caching. This means the software saves results so you do not have to process the same files twice. You find these results in the "Output" folder inside the main installation directory.

If you run out of space, click the "Clear Cache" button in the Settings menu. This deletes old data and frees up your hard drive. 

## 🔧 Troubleshooting Common Errors

Computers sometimes act in unexpected ways. Use these tips to solve common issues.

* The program does not start: Restart your computer and try again. Ensure you have the latest Windows updates installed.
* Processing takes too long: You selected a model that is too large for your hardware. Change the model to a smaller version in the settings menu.
* The error "Out of Memory" appears: Close other programs like web browsers or video players to free up your computer memory.
* Missing files: Make sure your input folder contains files in the correct format. The program only reads .txt and .csv files.

If you have trouble, check the logs. Click "View Logs" in the Help menu. This file contains a history of the actions the program performed. You can share this file if you ask someone else for help.

## 🎓 Understanding the Features

Q-ReAlign offers several tools to manage your text.

### Dataset Builders
This tool merges multiple text files into one. Use this if you have many small notes and want to process them in a single batch.

### Scoring
The scoring system ranks your text based on the instructions you provide. It assigns a number to each piece of text. Higher numbers indicate a closer match to your requirements. You can export these scores to a spreadsheet for better organization.

### CLI Workflows
Advanced users like to automate tasks. While this program has a visual menu, it also reads commands from a list. You can save your preferences as a command file and run the same task every week without clicking any buttons.
